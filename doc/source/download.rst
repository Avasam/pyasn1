Download PyASN1
===============

.. toctree::
   :maxdepth: 2

The PyASN1 software is provided under terms and conditions of BSD-style 
license, and can be freely downloaded from `Github <https://github.com/etingof/pyasn1>`_
or `PyPI <http://pypi.python.org/pypi/pyasn1/>`_.

The best way to obtain PyASN1 is by running `pip`:

.. code-block:: bash

   $ pip install pyasn1

or

.. code-block:: bash

   $ easy_install pyasn1

In case of any issues, please `let us know <https://github.com/etingof/pyasn1/issues/new>`_ so we could try to help out.
